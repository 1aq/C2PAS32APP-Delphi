New Update:

This is an updated version of the commandline tool using commandline pipes to parse the files which makes for an updated alternative. Open your files from within
any directory and the .pas file will be created in the same directory. Please note that the original file to be processed is copied to the core directory,upon completion of the processing,
the file is deleted and the newly created Pascal file is moved to the original .C, .CPP or .h directory.

This process was implimented to overcome the limitation of the tools design. Use the FileOpen option to Select the file to be be parsed.

The options are .c,.h and .cpp files to parse.

Original:<br>
c2pas32 is a command-line tool that translates a C files (*.h; *.c) to a pascal (Delphi4,5, VP, FPC) unit. It reads the C file and translates 80% of the C declarations (written in pure Win32 API) to equivalent pascal declarations. Additionally it format pascal code from destination file for better readability.

Syntax:
c2pas32.exe [filname]

Options (c2pas32.ini):
  • externlib=name.dll
    when this is specified, all function declarations are translated to
    function func(params); external name.dll name 'func';
    otherwise to
    function func(params);
  • fakebrakets=0
    removes brakets "()" from declarations of function without parameters.
  • usedoubleslash=0
    replases //... with (*...*)
